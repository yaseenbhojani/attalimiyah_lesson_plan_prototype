## Usage

1. **Open the Application**: Open `index.html` in a web browser
2. **Fill Lesson Details**: Start with basic lesson information
3. **Set Learning Objectives**: Define cognitive, psychomotor, and affective goals
4. **Plan Integration**: Tag cross-curricular connections
5. **Add Resources**: Upload or link teaching materials
6. **Design Instruction**: Plan anticipation, building knowledge, and consolidation
7. **Set Assessment**: Define evaluation methods and timeline
8. **Assign Homework**: Create homework assignments (post-lesson)
9. **Save & Submit**: Save drafts and submit for review
10. **Reflect**: Complete post-lesson reflection after teaching
